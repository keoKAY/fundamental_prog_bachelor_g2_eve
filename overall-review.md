> DRY -> Don't Repeat Yourself    

    - Structure: User-Defined Type 
```bash
struct Student{
    // data member 
    int id ; 
    string name; 
    string gender; 
    float score;
    // function members 
    // constructor  
};

```

**Structures** : 
- Grouping the data member and function members together inside one name. 
- Allow you to easily work with the data 
- There are three component of structure : 
    1. Data Member : variables used to store the data 
    2. Function Member: reusuable code that are repeatitive 
    3. Constructor : Special type of function used to provide values to the structure variables.

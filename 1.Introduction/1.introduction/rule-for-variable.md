## RULE FOR naming a variable 


1. Can start with letter (a-z,A-Z) or _, or contains number (0-9)
2. Cannot start with number 
3. Cannot contain whitespace 
4. cannot be a keyword 
5. symbol like %,#, .. cannot allowed 

Keyword: special words that has meaning for compiler 

---
### Convention for naming a variable
- name for variable or function , must write in camelCase 
- constant variable used UPPERCASE 
- short , concise 


lowercase = firstname; 
uppercase = FIRSTNAME; 
snakecase = first_name; 
camelcase = firstName;
